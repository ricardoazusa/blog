# 我的第一篇博客作业
使用Hugo + GitHub Actions + GitHub Pages搭建个人静态博客。
## 部署流程
1. 本地搭建Hugo项目，编写Markdown文章;
2. 推送源码至GitHub仓库，触发自动流水线;
3. Actions云端构建静态页面，自动发布到Pages站点.
